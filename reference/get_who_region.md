# Get WHO region

Get WHO region

## Usage

``` r
get_who_region(x)

get_who_regions(x)

get_region(x)

get_regions(x)

add_countries()
```

## Arguments

- x:

  A character value or vector of values of WHO countries to get the WHO
  region/s of.

## Value

A character value or vector of values of WHO regions.

## Examples

``` r
if (FALSE) { # \dontrun{
  get_who_regions("United Kingdom")
} # }
```
