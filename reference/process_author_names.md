# Process author names

Process author names

## Usage

``` r
process_author_names(authors)
```

## Arguments

- authors:

  A character vector of author names

## Value

A concatenated string of author names formatted as per person
specification.

## Examples

``` r
if (FALSE) { # \dontrun{
  process_author_names(c("John Doe", "Jane Doe"))
} # }
```
