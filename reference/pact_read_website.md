# Read datasets from the Pandemic PACT website

Read datasets from the Pandemic PACT website

## Usage

``` r
pact_read_website(.url = NULL)
```

## Arguments

- .url:

  The URL for the website dataset.

## Value

A tibble of the Pandemic PACT dataset from the website.

## Examples

``` r
if (FALSE) { # \dontrun{
  pact_read_website()
} # }
```
